TiltCheck Asset & Design Prompts

This document contains reusable prompts for generating visual assets via Canva or other design tools.

---

Style Guidelines

All TiltCheck visuals should reflect:

  • Minimal, professional aesthetic — no emoji-style clipart
  • Neon cyan/purple accents on deep charcoal (#0a0a0a)
  • Sharp geometric lines, thin glows, subtle noise grain (6–10%)
  • Dashboard/data-driven vibe — think industrial tech, not casino gloss
  • Clean typography — prefer Inter, IBM Plex Sans, or similar sans-serif
  • Non-glossy UI — avoid reflections, heavy gradients, or skeuomorphism

Canva Prompt 1: Hero Background Art

Title: The Degen Transparency Layer — Hero Background

Prompt:


Dark, professional landing hero background for a tech product called TiltCheck. 
Neon cyan and purple accent palette on deep charcoal (#0a0a0a). 

Visual concept: casino chips subtly dissolving into data streams or point-cloud lines, 
with thin trust gauges and vertical scanlines flickering online. 

No characters, no cartoon style, no logos, no text. 

Clean, minimal, high-contrast edges, subtle glow, noise grain at 6–10%. 

Composition: left/right negative space for headline and CTAs; focal energy centered 
with radial glow, thin grid lines and soft parallax blur. 

Style keywords: cyber-industrial, data-driven, non-glossy, editorial tech. 

Avoid stocky casino imagery. 

Final aspect: 16:9 and 21:9 variants.

Output: Full-width hero background (3840×2160 or 2560×1080 recommended)

Canva Prompt 2: Trust Gauges Component Set

Title: Trust Gauges — Dashboard UI Components

Prompt:


Create a UI component pack of "trust gauges" for a dashboard. 

Styles: circular gauge, horizontal bar, pill badge, and small inline sparkline — 
all in neon cyan/purple on charcoal with subtle glassy blur (very light). 

Clean ticks, data labels at 60% opacity, no skeuomorphism. 

Include states: stable, improving, declining, unknown. 

Provide light and dark variants. 

Export as transparent PNGs and SVGs. 

No brand marks, no text. 

Keep shapes geometric and minimal.

Output: Component set with 4 gauge types × 4 states = 16 assets

Canva Prompt 3: Tool Icons — Minimal Line Set

Title: TiltCheck Tool Icons (9-icon set)

Prompt:


Generate a 9-icon set in a consistent minimal line style, 2px stroke, rounded joins, 
cyan (#00d4aa) on charcoal (#0a0a0a). 

Subjects: 
1. Tipping (coins/transfer)
2. Swaps (exchange arrows)
3. Link scan (magnifying glass + chain)
4. Bonus clock (clock + gift)
5. Promo review (checklist + star)
6. Poker (cards)
7. Behavior/tilt (brain + alert)
8. Survey router (form + routes)
9. Games (controller/dice)

Each icon should be abstract but readable: think ISO-style pictograms for a dashboard. 

No emojis, no clipart, no gradients; allow tiny accent fills where necessary. 

Provide 512×512 and 128×128 exports, plus SVG, with solid and outline variants.

Output: 9 icons × 2 variants (solid, outline) = 18 files (PNG + SVG)

Canva Prompt 5: Trust Engine Diagram

Title: TiltCheck Trust Engines — Concept Diagram

Prompt:


Create a simple flow diagram showing two trust engines: Casino Trust Engine and Degen Trust Engine. 

Style: minimal, geometric blocks connected by thin arrows, neon cyan/purple on charcoal. 

Include labeled boxes for: 
- Casino inputs (payouts, bonuses, fairness)
- Degen inputs (behavior, accountability, scam detection)
- Central scoring node
- Output signals (alerts, scores, recommendations)

No illustrations, no people, no emoji. 

Export: horizontal layout, 1920×1080, transparent PNG + SVG.

Output: System architecture diagram for documentation or landing

|-------|-----|-------|

| Deep charcoal | #0a0a0a | Backgrounds |

| Secondary dark | #111 | Sections, cards |

| Card borders | #222 | Subtle dividers |

| Muted text | #888 | Secondary text |

| Primary text | #e0e0e0 | Body copy |

| Accent cyan | #00d4aa | CTAs, highlights |

| Accent purple | #00a8ff | Gradients, accents |

| Error red | #ff6b6b | Alerts, warnings |

| Warning yellow | #ffc107 | Beta badges |

Typography Reference

Headings:

  • Font: Inter Black / IBM Plex Sans Bold
  • Weight: 700–900
  • Style: Clean, sans-serif, high-contrast

Body:

  • Font: Inter Regular / IBM Plex Sans Regular
  • Weight: 400–500
  • Line height: 1.6–1.8

Code/Monospace:

  • Font: Courier New / IBM Plex Mono
  • Usage: Command examples, technical details

---

Notes for Designers

  • TiltCheck is not a casino — avoid slot machine, neon casino clichés
  • Visuals should feel trustworthy, transparent, and data-driven
  • Target audience: degens, mods, developers — smart, skeptical, detail-oriented
  • Brand voice: direct, clever, honest, zero BS
  • Avoid: corporate stock photos, emoji, cartoons, excessive gradients
