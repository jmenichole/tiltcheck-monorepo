Trust Gauges

Interactive gauge patterns live in components/trust-gauges.html. Dashboard integration adds dynamic arcs, reliability ratio, and volatility computation.

Guidelines:

  • Ring stroke width: 10px minimum for clarity on dark.
  • Animate arc changes < 800ms; ease not linear.
  • Label in uppercase 0.7rem with muted color (#888).
  • Never mix more than two accent hues per gauge cluster.

Principles

Clarity
No ornamental shadows; trust visuals must read instantly.
Restraint
Limited color usage preserves semantic impact (danger vs accent).
Non-Custodial
No interface suggesting holding funds or private keys.
Explainability
Every aggregate should map to underlying evidence accessible to users.
