Scam Protection

How does link scanning work?

When you post a URL in Discord, TiltCheck scans it via VirusTotal and checks our internal database of known phishing/scam domains. If it's sus, we block it and warn the channel.

What if a legitimate link gets blocked?

False positives are rare but possible. Server admins can whitelist specific domains. Report false positives to jme@tiltcheck.me.

How do I report a scam casino?

Email jme@tiltcheck.me with:

  • Casino domain/name
  • What happened (withdrawal denied, account locked, etc.)
  • Proof (screenshots, transaction IDs)

We'll investigate and update the trust score.

Troubleshooting

The bot isn't responding. What's wrong?

Possible issues:

  • Bot is offline for maintenance (check Discord status)
  • Bot lacks permissions (check role settings)
  • Bot was kicked from the server (re-invite it)

Why isn't tilt detection working?

TiltCheck only monitors channels it has access to. Make sure the bot can read messages in the channel you're using.

I got a false tilt warning. How do I adjust it?

Server admins can increase the tilt threshold via /tiltcheck settings. Default is 65/100.
